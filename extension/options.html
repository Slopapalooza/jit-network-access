Fastest way to enroll: open the registration link your admin sent you and click Enroll this device — nothing to copy. Use the form below only if you were handed a setup string instead.

Manual enrollment

Enrolling asks for permission to the listed sites so the extension can knock there. The secret is stored as a non-extractable key — treat the pasted string as burned afterward.

Setup string formats

Recommended (one-time code — the secret is fetched over TLS, never in the string):
jitaccess://enroll?v=1&server=https://app.example.com&code=<one-time-code>&origins=https://app.example.com&label=<name>
Also accepted (direct import): …&kid=<kid>&secret=<b64url>&origins=…
