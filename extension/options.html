JIT Network Access

Enroll a device token

Paste the setup string from your admin:
jitaccess://enroll?v=1&kid=<kid>&secret=<b64url>&origins=https://app.example.com&label=<name>

Enrolling asks for permission to the listed origins so the extension can knock there. The secret is imported as a non-extractable key and the pasted text should be treated as burned afterward.
